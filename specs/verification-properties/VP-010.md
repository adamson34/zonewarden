# VP-010: Multicast/Broadcast Exemption Detection (DI-016)

## Property Statement

`multicast::classify_dst(dst_ip: IpAddr, zone_match: Option<&ZoneMatch>) -> MulticastResult`
returns `MulticastBroadcast` **if and only if**:

**Step 1 (family-wide, no zone needed):**
- dst ∈ `224.0.0.0/4` (IPv4 multicast), OR
- dst == `255.255.255.255` (limited broadcast), OR
- dst ∈ `ff00::/8` (IPv6 multicast)

**Step 2 (directed broadcast, IPv4 only, requires zone match):**
- The zone matched via longest-prefix has a prefix length ≤ 30, AND
- dst equals the all-ones host address of that zone's IPv4 prefix, AND
- The prefix length is NOT /31 or /32 (those are excluded — DI-016)

For all other inputs, returns `NotMulticast` and sets `idmz_bypass` evaluation to proceed normally.

The function is total (no panics) and two-step sequential (no circularity).

## Source Contract

- **BC:** BC-1.03.003 — Multicast/broadcast destination short-circuits to MulticastBroadcast
- **BC:** BC-1.03.004 — Directed-broadcast destination override for ≤/30 zones
- **Postcondition/Invariant:** DI-016 (multicast/broadcast exemption + verdict precedence)
- **Also covers:** BC-1.04.011 (MulticastExempt short-circuits before IntraZone and conduit evaluation)

## Proof Harness Skeleton

```rust
#[cfg(kani)]
#[kani::proof]
fn vp010_multicast_step1_ipv4() {
    let addr: u32 = kani::any();
    let ip = Ipv4Addr::from(addr);

    // Step 1 boundaries:
    let in_multicast_range = (addr >> 28) == 14; // 224.0.0.0/4
    let is_limited_bcast = addr == u32::MAX;     // 255.255.255.255

    let result = multicast::classify_dst(IpAddr::V4(ip), None);

    if in_multicast_range || is_limited_bcast {
        assert_eq!(result, MulticastResult::MulticastBroadcast);
    } else {
        assert_eq!(result, MulticastResult::NotMulticast);
    }
}

#[cfg(kani)]
#[kani::proof]
fn vp010_directed_bcast_excluded_for_slash31_slash32() {
    // A /31 or /32 must never be classified as directed broadcast
    let prefix_len: u8 = kani::any();
    kani::assume(prefix_len == 31 || prefix_len == 32);

    let net_addr: u32 = kani::any();
    // The "broadcast" (all-ones host) address for this "network"
    let all_ones_host: u32 = net_addr | ((1u32 << (32 - prefix_len as u32)) - 1);
    let dst = Ipv4Addr::from(all_ones_host);

    // Must NOT classify as multicast (step 2 excluded for /31 and /32)
    let zone_match = ZoneMatch::explicit_ipv4(net_addr, prefix_len);
    let result = multicast::classify_dst(IpAddr::V4(dst), Some(&zone_match));

    // Not in multicast range, not limited broadcast, step 2 excluded → NotMulticast
    let in_mcast = (all_ones_host >> 28) == 14;
    let is_limited = all_ones_host == u32::MAX;
    if !in_mcast && !is_limited {
        assert_eq!(result, MulticastResult::NotMulticast);
    }
}
```
