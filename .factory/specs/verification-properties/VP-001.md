# VP-001: Total Endpoint Resolution (DI-003)

## Property Statement

For every IP address and every valid (tie-free, sorted) `PrefixIndex`, `resolver::resolve`
returns exactly one outcome: a declared zone ID, the sentinel `EXTERNAL` zone ID, or
(for multicast/broadcast destinations, handled by `multicast` before `resolve` is called)
the `MulticastBroadcast` match kind. The function never returns `None`, never panics,
and never silently drops an endpoint.

Formally: `∀ ip: IpAddr, ∀ index: PrefixIndex where is_valid(index) → resolve(index, ip) ≠ ⊥`

## Source Contract

- **BC:** BC-1.03.001 — Resolve endpoint to exactly one zone via longest-prefix match
- **Postcondition/Invariant:** DI-003 (total endpoint resolution); DI-004 (deterministic
  overlap resolution); DI-005 (implicit EXTERNAL fallback)
- **Also covers:** BC-1.03.002 (EXTERNAL fallback), BC-1.03.005 (both-EXTERNAL → IntraZone)

## Proof Harness Skeleton

```rust
#[cfg(kani)]
mod proofs {
    use super::*;

    #[kani::proof]
    #[kani::unwind(512)]
    fn vp001_total_resolution() {
        // Precondition: a valid (sorted, tie-free) prefix index with bounded size
        let len: usize = kani::any();
        kani::assume(len <= 32); // bounded for proof termination

        let index = PrefixIndex::kani_any_valid(len);
        // is_valid() asserts sorted descending by prefix_len, no ties in same family
        kani::assume(index.is_valid());

        let ip: IpAddr = kani::any();

        // Execute resolution
        let result = resolver::resolve(&index, ip);

        // Postcondition: always returns Some(ResolvedEndpoint)
        assert!(result.zone_id.is_some() || result.zone_id == ZoneId::EXTERNAL);
        // The result is one of: Explicit{prefix_len}, ImplicitExternal
        // (MulticastBroadcast is pre-checked by multicast module before resolve is called)
    }
}
```
