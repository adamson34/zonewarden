# VP-003: IDMZ No-Bypass Rule (DI-006)

## Property Statement

`idmz::check_bypass(src_endpoint, dst_endpoint) -> bool` returns `true` **if and only if**:
- Both endpoints resolve to managed (non-EXTERNAL) zones.
- Neither endpoint is in the IDMZ (PurdueLevel::Idmz).
- One endpoint's zone level is ≤ L3 (OT side).
- The other endpoint's zone level is ≥ L4 (IT side).
- Neither endpoint is `MatchKind::MulticastBroadcast`.

For all other input combinations, the function returns `false`. The function is total
(no panics, no unreachable arms).

## Source Contract

- **BC:** BC-1.04.007 — IDMZ no-bypass additive finding (DI-006)
- **BC:** BC-1.04.008 — EXTERNAL and MulticastBroadcast exclusion
- **Postcondition/Invariant:** DI-006 truth table (all 9 rows)

## Proof Harness Skeleton

```rust
#[cfg(kani)]
#[kani::proof]
fn vp003_idmz_bypass_correctness() {
    let src_level: PurdueLevel = kani::any();
    let dst_level: PurdueLevel = kani::any();
    let src_is_external: bool = kani::any();
    let dst_is_external: bool = kani::any();
    let src_is_multicast: bool = kani::any();
    let dst_is_multicast: bool = kani::any();

    let src = ResolvedEndpoint::new_kani(src_level, src_is_external, src_is_multicast);
    let dst = ResolvedEndpoint::new_kani(dst_level, dst_is_external, dst_is_multicast);

    let result = idmz::check_bypass(&src, &dst);

    // Verify against truth table
    let expected = idmz_truth_table(src_level, src_is_external, src_is_multicast,
                                     dst_level, dst_is_external, dst_is_multicast);
    assert_eq!(result, expected);

    // Specific invariants from DI-006:
    // EXTERNAL endpoint → never bypass
    if src_is_external || dst_is_external {
        assert!(!result);
    }
    // MulticastBroadcast → never bypass (DI-016)
    if src_is_multicast || dst_is_multicast {
        assert!(!result);
    }
    // IDMZ endpoint → never bypass
    if src_level == PurdueLevel::Idmz || dst_level == PurdueLevel::Idmz {
        assert!(!result);
    }
}
```
