# VP-009: Offline Purity — No Network I/O in Core (DI-012)

## Property Statement

The `zonewarden-core` crate never opens a network socket, never performs DNS resolution,
and never makes any network I/O call, at any point during execution. This is enforced
structurally: the core lib crate has no dependency on `std::net`, `tokio::net`, or any
network-capable crate (direct or transitive).

Additionally, the core crate never mutates its input data (read-only invariant, DI-012).

## Source Contract

- **BC:** BC-1.06.007 — zonewarden never opens a network socket or mutates input files
- **Postcondition/Invariant:** DI-012 (read-only / offline)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| Structural (manual) | `cargo deny` + `cargo tree` + compile-time `#![forbid]` | N/A | All transitive deps of `zonewarden-core` |

**Note:** This property is NOT a Kani target (Kani cannot reason about syscall absence
from outside the process). It is enforced by:
1. `cargo deny` in CI: `deny = ["net"]` capability blocks network-capable deps.
2. `zonewarden-core/src/lib.rs`: `#![forbid(unsafe_code)]` prevents `unsafe extern "C"`.
3. CI step: `cargo tree -p zonewarden-core | grep -E 'tokio|reqwest|hyper|net2'` must
   return empty.
4. `strace`/`dtrace` integration test in CI: zero `socket()` syscalls during a run.

The read-only invariant is enforced by opening input files with `O_RDONLY` (verified
by inspection of `policy::load` and `zeek_adapter::new`).

## Proof Harness Skeleton

```yaml
# .github/workflows/ci.yml (offline enforcement steps)
- name: Verify no network deps in core
  run: |
    cargo tree -p zonewarden-core --no-dedupe | \
      grep -E 'tokio|reqwest|hyper|net2|socket2' && \
      echo "FAIL: network dep found in core" && exit 1 || \
      echo "PASS: no network deps"

- name: Verify no network syscalls (Linux)
  run: |
    strace -e trace=network \
      ./target/release/zonewarden \
      --policy tests/fixtures/basic-policy.yaml \
      --flows tests/fixtures/empty.log \
      2>&1 | grep -v "^strace:" && \
      echo "FAIL: network syscall detected" && exit 1 || \
      echo "PASS: no network syscalls"
```
