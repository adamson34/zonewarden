# VP-005: Accounting Identity and Bounds (DI-015)

## Property Statement

For any sequence of `Verdict` values processed by `aggregator::aggregate`:

1. `total_flows == intra_zone + allowed + no_matching_conduit + wrong_direction + multicast_exempt`
2. `distinct_violating_flows ≤ total_flows`
3. `idmz_bypasses ≤ total_flows`
4. `skipped` is excluded from `total_flows` (counted separately)
5. All tallies use `u64`; `checked_add` is used; overflow returns `Err(SysError::TallyOverflow)`

The accounting identity holds for every valid input, including the edge cases:
- Empty flow sequence: all tallies 0, identity holds trivially.
- All flows intra-zone: `total_flows == intra_zone`.
- All flows violated: `total_flows == no_matching_conduit` (or wrong_direction, etc.).

## Source Contract

- **BC:** BC-1.05.001 — DI-015 accounting identity holds for every ConformanceResult
- **Postcondition/Invariant:** DI-015 (verdict totality and accounting)
- **Also covers:** BC-1.05.004 (u64 overflow → exit 2), BC-1.05.005 (empty input → all-zero)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | proptest (property-based testing) | No bound needed | Random sequences of VerdictKind; overflow edge cases |

**Note:** Kani is not used here because `aggregate` processes an iterator of variable
length (up to `max_flows`). proptest generates sequences of 0..10,000 verdicts and
asserts the identity on each. The overflow path (VP-006) is a separate Kani target.

## Proof Harness Skeleton

```rust
use proptest::prelude::*;

proptest! {
    #[test]
    fn vp005_accounting_identity(
        verdicts in prop::collection::vec(arb_verdict(), 0..10_000)
    ) {
        let result = aggregator::aggregate(
            verdicts.iter().cloned(),
            &dummy_policy(),
            0, // skipped
        ).expect("aggregate should not overflow for reasonable inputs");

        let sum = result.intra_zone
            .checked_add(result.allowed).unwrap()
            .checked_add(result.no_matching_conduit).unwrap()
            .checked_add(result.wrong_direction).unwrap()
            .checked_add(result.multicast_exempt).unwrap();

        prop_assert_eq!(sum, result.total_flows);
        prop_assert!(result.distinct_violating_flows <= result.total_flows);
        prop_assert!(result.idmz_bypasses <= result.total_flows);
    }

    #[test]
    fn vp005_empty_input_all_zero(_: ()) {
        let result = aggregator::aggregate(
            std::iter::empty(),
            &dummy_policy(),
            0,
        ).unwrap();
        prop_assert_eq!(result.total_flows, 0);
        prop_assert_eq!(result.intra_zone, 0);
        // ... all fields zero
    }
}
```
