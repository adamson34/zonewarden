# VP-008: PortSet Canonical Form — Sorted, Non-Overlapping, Non-Adjacent (DI-020)

## Property Statement

`portset::canonicalize(ranges: &[PortRange]) -> PortSet` produces a `PortSet::Ranges(v)`
where `v` satisfies:
1. Sorted in ascending order by `lo`.
2. No two ranges overlap (for all i: `v[i].hi < v[i+1].lo`).
3. No two ranges are adjacent (for all i: `v[i].hi + 1 < v[i+1].lo`).
4. Adjacent ranges in the input are coalesced (e.g., `[500-502, 503]` → `[500-503]`).
5. Overlapping ranges in the input are merged.
6. `PortSet::Any` is a distinct sentinel and is never produced by `canonicalize`
   (it is parsed separately from the literal `"any"` token).

Additionally: `portset::matches_port(ps: &PortSet, port: Option<u16>) -> bool` returns
`true` iff port is contained in any range in `ps`, or `ps == PortSet::Any`.
`PortSet::Any` also matches `port: None` (portless protocols); an explicit range set
never matches `port: None`.

## Source Contract

- **BC:** BC-1.01.009 — PortSet canonical form: sorted non-overlapping non-adjacent ranges
- **Postcondition/Invariant:** DI-020 (PortSet canonical form)
- **Also covers:** BC-1.04.006 (portless protocol matches only `ports: Any`)

## Proof Harness Skeleton

```rust
#[cfg(kani)]
#[kani::proof]
#[kani::unwind(32)]
fn vp008_portset_canonical_form() {
    // Bounded input: up to 4 ranges
    let n: usize = kani::any();
    kani::assume(n <= 4);

    let mut ranges: Vec<PortRange> = (0..n)
        .map(|_| {
            let lo: u16 = kani::any();
            let hi: u16 = kani::any();
            kani::assume(lo <= hi);
            PortRange { lo, hi }
        })
        .collect();

    let result = portset::canonicalize(&ranges);

    if let PortSet::Ranges(v) = &result {
        // Sorted
        for i in 0..v.len().saturating_sub(1) {
            assert!(v[i].lo < v[i+1].lo);
        }
        // Non-overlapping
        for i in 0..v.len().saturating_sub(1) {
            assert!(v[i].hi < v[i+1].lo);
        }
        // Non-adjacent (gap of at least 2)
        for i in 0..v.len().saturating_sub(1) {
            assert!(v[i].hi + 1 < v[i+1].lo);
        }
        // Coverage: every original port is in the result
        for r in &ranges {
            for p in r.lo..=r.hi {
                assert!(portset::matches_port(&result, Some(p)));
            }
        }
    }
}

#[cfg(kani)]
#[kani::proof]
fn vp008_portless_any_matches_none() {
    assert!(portset::matches_port(&PortSet::Any, None));
}

#[cfg(kani)]
#[kani::proof]
fn vp008_explicit_range_does_not_match_none() {
    let lo: u16 = kani::any();
    let hi: u16 = kani::any();
    kani::assume(lo <= hi);
    let ps = PortSet::Ranges(vec![PortRange { lo, hi }]);
    assert!(!portset::matches_port(&ps, None));
}
```
